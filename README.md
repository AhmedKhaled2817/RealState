# 🏠 RealState - Modern Real Estate Website

A stylish, responsive, and user-friendly real estate website designed to provide an immersive experience for users looking to explore property listings. Built using clean **HTML5**, **CSS3**, and **JavaScript**, this project simulates a professional property showcase ideal for real-world real estate platforms.

## ✨ Features

- ✅ **Responsive Layout**: Optimized for all screen sizes — mobile, tablet, and desktop.
- 🏘️ **Property Showcase**: Clean and organized display of featured properties.
- 🎯 **Navigation & Sections**: Smooth scrolling and clearly divided sections.
- 🌙 **Dark Mode**: Toggle between light and dark themes for user comfort.
- 🖱️ **Scroll Animations**: Engaging effects using ScrollReveal.js.
- 🧩 **Reusable Components**: Clean and maintainable code for future scalability.

---

## 🛠️ Technologies Used

| Technology        | Description                                                        |
|-------------------|--------------------------------------------------------------------|
| `HTML5`            | Markup and semantic structure                                     |
| `CSS3`             | Layout, animations, responsiveness                                |
| `JavaScript (ES6+)`| Interactivity and DOM manipulation                               |
| `Swiper.js`        | Sliders and carousels for modern UI                              |
| `ScrollReveal.js`  | Scroll-based animation effects                                   |
